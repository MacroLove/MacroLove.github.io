少林罗汉拳 预备一二三势 童子拜佛和马步单鞭和展脚冲拳

预备势

两脚并步站立，身胸挺直，两臂自然下垂.两掌附于大腿外侧，掌心向里。目视前方(图1)

接上动作，左脚向左开半步，两掌由下向上、向外、往内划弧，然后变拳抱于腰间。目视左侧(图2 )



童子拜佛

抬左腿向右提膝，盘于右膝上。同时两手由下向外、向上划弧，然后再由外向内屈肘合掌.掌心相对，掌指向上，高与胸平。目视前方(图3 )


马步单鞭

左脚向左落一步，两腿屈膝半蹲，使两腿成马步。同时两手由内向两侧展臂摆掌，两掌心向外。目视左侧(图4 )

展脚冲拳

以两脚为轴，体向左转90渡，左腿屈膝，右腿蹬直，使两腿成左弓步。同时左掌抖腕变拳，右掌展臂向右。目视左拳(图5 )

接上动作，抬右脚向前上一步，震脚，与左脚成并步，两腿半蹲。同时右掌变拳，随身向前屈肘冲击;左拳向内屈肘，护于右肘下。目视右拳(图6 )
