醉拳 转身云杯( 太白转身右敬酒 )

重心左移，体左转。双脚前掌左外碾地135度成左屈膝踏实，右脚后跟抬起，前掌着地成半跪步。与此同时，右手随体左转由右侧前上方向下、过体档前向左上方顺时针划弧云杯敬酒，位于转体后的右前方，与肩同高。体成右半跪步右云杯敬酒势，目视右手。(图68)

要点，拧腰、摆胯、侧肩，云杯有力。

醉拳浪叉步斟酒势( 何仙姑侧进身偏 )

①右提膝摆盖杯:重心在左，休上升。左脚左外碾地30度成直腿踏实。右脚随左外碾地由右后下方提膝，脚背绷直。与此同时，右手由右前方向上、向休左胸前转向下、经腹前向右顺时针划弧摆按杯。左手由左腰侧向左侧后、向上、转向前逆时针划弧摆杯，高于头。体成左独立右按左盖杯势，目视左手。(图69 )

要点：提气、收腹、顶膝，按、盖杯快速有力。

②右醉浪步盖杯。上动不停，重心在左，体下蹲。左脚左内碾地15度成屈膝，后跟抬起，前掌着地。右脚随内碾地向左脚前方浪沧醉落一步(踩踏，攻其敌方下部)，脚尖内扣，屈膝踏实。与此同时，左手由左前上方向前、向下、向划弧盖杯，与肩同高。体成右醉步左盖杯势，目视左手。(图70 )

要点:沉肩、沉胯，盖杯快速有力。

③左浪右叉斟酒势:上动不停，上下同动，重心在右，体成叉步。右脚右外碾地30度，左脚随右外碾地由左后上方向右脚前浪沧醉进一步(脚外侧踩踏，攻其敌方下部).脚尖外撇，屈膝踏实。右脚在左脚踏实后即刻向左脚后方外侧浪沧醉叉一步(脚内侧踩踏)，后跟抬起，前掌着地。与此同时，左手由左前方向前划弧摆杯，高于有。右手随右叉步由右腰侧变八字指向右侧后、向上、过头顶向下逆时针划弧摆指斟酒，微屈肘。体成叉步斟酒势，目视左手。(图71)

要点:拧腰、侧肩、摆杯、摆指迅速有力。
